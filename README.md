# Task Manager API

Structured REST API built with ASP.NET Core.

## Overview
This project demonstrates how to build a clean and organized backend API using service-layer architecture and dependency injection.

## Features
- Create tasks
- Retrieve all tasks
- Retrieve task by ID
- Delete tasks
- In-memory data storage

## Architecture
- Controllers
- Service layer
- Dependency Injection

## Tech Stack
- C#
- .NET
- ASP.NET Core Web API
